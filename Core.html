POPULAR SEMANTIC TAGS

Tag Description
<header> Represents introductory content or a container for navigational links.
<nav> Defines a section that contains navigation links.
<main> Represents the main content of a document, excluding headers, footers, and sidebars.
<article> Represents self-contained content, such as a blog post or news article.
<section> Groups related content together within a document.
<footer> Represents footer content for a section or document, usually containing metadata, copyright info, etc.
